Installation¶

Python¶

Install from GitHub: pip install git+https://github.com/munichpavel/fake-data-for-learning

For local development

  • git clone the repository and cd into the root directory

  • Create a virtual environment from the included requirements.txt file

Documentation¶

To generate your own Sphinx documentation, you must set the environment variable LOCAL_BUILDDIR.

Convenience scripts for the case of a separate build directories (locally and remotely) are in docs/scripts of the GitHub repository.
